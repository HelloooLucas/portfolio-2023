Last Quest

Last Quest is a six episodes long comics project initiated by Camille Prieur and Vincent Malgras, two French writers and comics artists. It was my first freelance project, started even before i finished my coding bootcamp

The website is fully responsive and consists of a landing page redirecting to six pages, each hosting one episode of the series

Each episode is enriched with animated gif images, sound effects and the possibility to set the page on full screen

Parallax and gif timeline control on scroll

The scroll-based animations are achieved using the greensock animation platform, a very powerful and efficient tool
