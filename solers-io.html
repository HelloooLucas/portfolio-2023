Solers.io

Solers.io is a job board for developers, that takes a twist at skills assertion. once a developer has declared their skills, they can pass online certifications that would act as proof for recruiters and remove the need for further technical testing

The website is fully responsive, allows developers to create an account, declare skills, pass certifications and apply to job offers. I did only the frontend part of the project
